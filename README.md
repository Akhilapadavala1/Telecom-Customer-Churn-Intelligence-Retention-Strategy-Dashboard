# Telecom Customer Churn Intelligence & Retention Strategy Dashboard

## Project Overview

This project analyzes telecom customer data to identify customer churn patterns, high-risk customers, and retention opportunities using SQL, Python, and Power BI.

## Tools Used

* SQL Server
* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Power BI
* CSV Dataset

## Project Objectives

* Analyze customer churn behavior
* Create customer risk scores
* Identify heavy usage customers
* Perform exploratory data analysis (EDA)
* Build an interactive Power BI dashboard
* Generate business insights for retention strategies

## SQL Tasks

* Customer Analysis
* Churn Rate Calculation
* Revenue Analysis
* Contract Type Analysis
* Window Functions
* High-Risk Customer Identification

## Python Tasks

* Customer Risk Score using Functions and If-Else
* Heavy Usage Flag using Lambda Functions
* Exploratory Data Analysis (EDA)
* Data Visualizations

## Dashboard Features

* Total Customers
* Churn Rate
* Revenue Analysis
* Customer Segmentation
* Contract Analysis
* Retention Insights

## Files Included

* Telecom_Customer_Churn_Analysis.ipynb
* telecom_sql_queries.sql
* Telecom_Dashboard.pbix
* Telecom churn flag data.csv
* dashboard1.png
* dashboard2.png

## Key Insights

* Customers with shorter tenure are more likely to churn.
* Month-to-month contracts show higher churn rates.
* High-value customers should be targeted for retention programs.
* Customer segmentation helps improve retention strategies.
